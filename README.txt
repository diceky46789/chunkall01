Chunk Flashcard Pro v1.0

GitHub Pagesで使う方法:
1. このZIPを展開します。
2. index.html, style.css, app.js, template_chunk_flashcards.csv, sample_chunk_flashcards.csv をGitHubリポジトリにアップロードします。
3. Settings → Pages で公開します。
4. アプリを開き、CSVを読み込みます。
5. 保存済みCSVに表示された「このCSVで練習」を押します。

v1.0の機能:
- CSV読み込み
- CSVを端末内に保存
- 保存済みCSV一覧
- CSVごとの練習開始
- 英語→日本語 / 日本語→英語
- チャンク種類別フィルター
- タップで答え表示
- 前へ / 次へ
- 単語・熟語・解説表示
