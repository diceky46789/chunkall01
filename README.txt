Chunk Flashcard Pro v2.0

使い方
1. このフォルダ内の全ファイルをGitHubリポジトリにアップロードします。
2. GitHub Pagesを有効にします。
3. アプリを開き、CSVを追加します。
4. 保存済みCSVの「このCSVで練習」を押します。
5. 英語→日本語、日本語→英語、チャンク種類、表示順を選んで練習します。

v2.0の主な機能
- CSV読み込み
- CSVライブラリ保存
- CSV削除
- CSVごとの練習開始
- フラッシュカード表示
- タップで裏返し
- 英語→日本語 / 日本語→英語
- 前へ / 次へ / 最初 / 最後
- 左右スワイプ
- カード番号と進捗バー
- チャンク種類別フィルター
- シャッフル表示
- サンプルCSV追加
